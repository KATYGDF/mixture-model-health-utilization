Simulação de Dados de Utilização em Saúde Suplementar

Estudo preliminar · Pôster científico · v3.0

Este projeto gera uma amostra sintética de beneficiários de planos de saúde usando um modelo de mistura finita com grupos latentes de utilização. Os parâmetros foram calibrados para reproduzir indicadores oficiais do Painel de Indicadores da ANS.

Nenhum dado real de beneficiários foi utilizado — todos os dados são inteiramente simulados.

Destaques dos Resultados (v3.0):
✓ Calibração perfeita (desvios < 3% vs. Painel ANS)
✓ 4 grupos latentes com padrões heterogêneos
✓ 5.000 beneficiários sintéticos com características realistas
✓ Indicadores de alerta capturando anomalias esperadas

O modelo em resumo

Amostra 5.000 beneficiários simulados
Grupos latentes 4 grupos com proporções fixas na população
Desfechos de contagem Consultas · Pronto-socorro · Exames · Terapias · Internações
Distribuições Binomial Negativa NB₂ / ZINB / Beta-Binomial
Calibração Indicadores ANS — desvio < 3% em todos os desfechos
Implementação R · dplyr · ggplot2 · MASS

Os quatro grupos latentes

Cada beneficiário pertence a exatamente um grupo latente gi ∈ {1, 2, 3, 4}, com proporções fixas π = (0,50; 0,25; 0,20; 0,05).

Grupo Nome Proporção Perfil de utilização
G1 Baixo Uso 50% ~3 consultas/ano · ~2 exames · <5% internação
G2 Ambulatorial Coordenado 25% ~8 consultas/ano · ~10 exames · alta taxa de terapias
G3 Agudo / Hospitalar 20% ~5 consultas/ano · ~3 PS/ano · ~80% com internação
G4 Padrão Atípico 5% ~4 PS/ano · ~18 exames · razão exame/consulta ↑↑
