Modelagem de Heterogeneidade em Dados de Saúde Suplementar via Modelos de Mistura

Estudo preliminar · Pôster científico · Doutorado

Este projeto investiga a estrutura latente de utilização de serviços de saúde por beneficiários de planos privados, utilizando modelos de mistura finita para identificar perfis heterogêneos de uso. Os dados são inteiramente sintéticos, gerados com parâmetros calibrados a indicadores oficiais do Painel de Indicadores da ANS.

A hipótese central é que modelos probabilísticos — em particular a mistura de Binomial Negativa — são mais capazes de recuperar grupos latentes em dados de contagem com sobredispersão do que métodos de clusterização geométrica como o K-means.

Etapas do projeto

Etapa 1 Simulação dos Dados

Geração de uma amostra sintética de 5.000 beneficiários por meio de uma mistura finita com quatro grupos latentes de utilização (baixo uso, ambulatorial coordenado, agudo/hospitalar e padrão atípico). Os parâmetros foram calibrados iterativamente para reproduzir indicadores do Painel ANS com desvio inferior a 3%.

Etapa 2 Análise Exploratória

Caracterização descritiva dos dados simulados: distribuição dos desfechos de contagem por grupo latente, proporção de zeros, sobredispersão, correlações e indicadores de alerta. Verificação da qualidade da simulação e identificação das variáveis com maior poder discriminativo entre os grupos.

Etapa 3 Modelagem

Comparação de três métodos não supervisionados para recuperação dos grupos latentes: mistura gaussiana, mistura de Binomial Negativa (algoritmo EM implementado do zero) e K-means como baseline geométrico. A qualidade de recuperação é avaliada por ARI, NMI e acurácia após pareamento ótimo via algoritmo húngaro.

Nenhum dado real de beneficiários foi utilizado — todos os dados são inteiramente simulados. · Referência metodológica: Ng, Xiang & Yau (2019), Mixture Modelling for Medical and Health Sciences, Chapman & Hall/CRC.
