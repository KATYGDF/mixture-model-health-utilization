Modelagem de Heterogeneidade em Dados de Saúde Suplementar via Modelos de Mistura

Estudo preliminar · Pôster científico · Doutorado

Este projeto investiga se modelos probabilísticos de mistura — em especial a Mistura de Binomiais Negativas — são mais adequados do que métodos geométricos, como o K-means, para recuperar grupos latentes em dados de contagem sobredispersos de utilização em saúde suplementar. Os dados são inteiramente sintéticos, gerados com parâmetros calibrados a indicadores oficiais do Painel de Indicadores da ANS; os grupos verdadeiros são conhecidos apenas porque a base foi simulada.

O objetivo é avaliar, em ambiente controlado, a capacidade de recuperação de estrutura latente de heterogeneidade em perfis de utilização — comparando a abordagem probabilística baseada em mistura NB com o baseline geométrico K-means. A saúde suplementar é o contexto substantivo; a aplicação operacional não é o escopo deste estudo.

Escopo do estudo. Este trabalho não tem como objetivo detectar fraude. Seu foco é avaliar, em ambiente controlado, se uma abordagem probabilística baseada em mistura NB recupera melhor a estrutura latente de perfis de utilização do que o K-means. Não há dados reais de beneficiários — todos os dados são inteiramente simulados.

Etapas do projeto

Etapa 1 Simulação dos Dados

Geração de uma amostra sintética de 5.000 beneficiários por meio de uma mistura finita com quatro grupos latentes de utilização (baixo uso, ambulatorial coordenado, agudo/hospitalar e padrão atípico). Os parâmetros foram calibrados iterativamente para reproduzir indicadores do Painel ANS com desvio inferior a 10%.

Etapa 2 Análise Exploratória

Caracterização descritiva dos dados simulados: distribuição dos desfechos de contagem por grupo latente, proporção de zeros, sobredispersão e correlações entre variáveis. Verificação da qualidade da simulação e identificação das variáveis com maior poder discriminativo entre os grupos não observados.

Etapa 3 Modelagem

Comparação de dois métodos não supervisionados para recuperação dos grupos latentes: mistura de Binomial Negativa (algoritmo EM implementado do zero) e K-means como baseline geométrico. A qualidade de recuperação é avaliada por ARI, NMI e acurácia após pareamento ótimo via algoritmo húngaro.

Nenhum dado real de beneficiários foi utilizado — todos os dados são inteiramente simulados. · Referência metodológica: Ng, Xiang & Yau (2019), Mixture Modelling for Medical and Health Sciences, Chapman & Hall/CRC.
