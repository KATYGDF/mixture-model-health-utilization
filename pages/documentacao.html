Passo a passo
da simulação

Descrição detalhada de cada etapa do script R, com raciocínio por trás de cada decisão de modelagem.

Esta documentação descreve a versão 3.0 do script — a versão final calibrada por indicadores ANS. Para o histórico de mudanças entre versões, veja a página Evolução.

Visão geral do fluxo

A simulação segue um fluxo sequencial em 16 etapas, organizado do mais simples ao mais complexo:

  1. Configuração inicial

    Carregamento dos pacotes necessários (dplyr, ggplot2, scales) e definição da semente aleatória para reprodutibilidade:

    set.seed(1234)

    A semente garante que qualquer pessoa que execute o script obtenha exatamente os mesmos dados, essencial para um pôster científico.

  2. Tamanho da amostra

    $n = 5.000$ beneficiários — suficiente para estabilizar as estimativas dos grupos menores (G4 = 5% → ~250 indivíduos) sem tornar o script lento.

  3. Atribuição de grupo latente

    Cada beneficiário recebe um grupo latente $g_i \in \{1,2,3,4\}$ amostrado de uma distribuição categórica com proporções $\boldsymbol{\pi} = (0{,}50,\, 0{,}25,\, 0{,}20,\, 0{,}05)$:

    grupo_latente <- sample(x = 1:4, size = n, replace = TRUE,
                        prob = pi_grupos)

    Este é o passo fundamental — todos os demais parâmetros dependem do grupo.

  4. Geração vetorizada das idades

    A distribuição de idades é uma mistura de três faixas. A implementação vetorizada (sem sapply) é ordens de magnitude mais eficiente para amostras grandes:

    faixa_idx <- sample(1:3, size = n, replace = TRUE,
                    prob = c(0.20, 0.55, 0.25))

idade[faixa_idx == 1] <- sample(0:17, sum(faixa_idx==1), replace=TRUE)
idade[faixa_idx == 2] <- sample(18:59, sum(faixa_idx==2), replace=TRUE)
idade[faixa_idx == 3] <- sample(60:85, sum(faixa_idx==3), replace=TRUE)

    A variável idade10 = idade/10 é usada nos modelos para que os coeficientes reflitam o efeito por década de vida.

  5. Demais covariáveis

    As covariáveis restantes são geradas independentemente do grupo latente (covariáveis observadas — o grupo é latente, não observável):

    • sexo — $\text{Bernoulli}(0{,}55)$
    • tempo_no_plano — $\text{Uniforme}(1, 120)$ meses
    • copart — $\text{Bernoulli}(0{,}40)$
    • regiao — $\text{Categórica}(0{,}40, 0{,}35, 0{,}25)$, codificada em dummies regiao2 e regiao3
  6. Interceptos por grupo (seção 4)

    Os interceptos $\alpha_{gk}$ definem a média-base de cada desfecho para cada grupo, no log da escala. Foram calibrados iterativamente para aproximar os alvos ANS:

    alpha_consultas <- log(c(2.5, 8.0, 4.0, 1.5)) - 0.588

    O ajuste aditivo (ex: - 0.588) é um deslocamento global aplicado após o log() para compensar o efeito médio das covariáveis na média populacional.

  7. Matrizes de coeficientes grupo-específicos (seção 5)

    Cada desfecho possui uma matriz $4 \times 6$ de coeficientes, onde as linhas correspondem aos grupos e as colunas às covariáveis:

    betas$consultas <- matrix(
  c(# idade10  sexo   tempo  copart  regiao2  regiao3
     0.08,    0.12,  0.03, -0.18,   0.05,    0.00,  # G1
     0.10,    0.14,  0.04, -0.10,   0.05,    0.00,  # G2
     0.07,    0.10,  0.02, -0.05,   0.05,    0.00,  # G3
    -0.05,    0.08,  0.01, -0.02,   0.00,    0.05), # G4
  nrow=4, byrow=TRUE)

    Decisões de modelagem embutidas:

    • copart tem coeficiente negativo em todos os grupos — coparticipação reduz utilização (efeito preço)
    • G4 tem coeficiente negativo para idade10 em consultas — padrão atípico com jovens de alto uso
    • G3 tem maior sensibilidade de idade para internações — condizente com envelhecimento e doença crônica
  8. Parâmetros de dispersão $\theta$

    O parâmetro $\theta$ da NB₂ controla a superdispersão. Internações têm $\theta = 0{,}8$ (alta dispersão) porque eventos raros e muito concentrados em poucos indivíduos geram variância muito maior que a média.

  9. Probabilidades de zero estrutural $\psi_{gk}$

    Para terapias e internações, definidas por grupo. O G1 (baixo uso) tem $\psi_{g=1,\,\text{internacoes}} = 0{,}97$ — 97% dos indivíduos desse grupo nunca seriam internados no período de referência, independentemente de quanto seus parâmetros NB permitiriam.

  10. Cálculo das médias condicionais $\mu_{igk}$

    Função vetorizada que calcula $\mu_{igk} = \exp(\alpha_{gk} + \mathbf{x}_i^\top \boldsymbol{\beta}_{gk})$ para todos os $n$ indivíduos simultaneamente, usando multiplicação elemento a elemento de matrizes:

    calc_mu <- function(alpha_vec, beta_mat, grupo, ...) {
  X <- cbind(idade10, sexo, tempo_anos, copart, regiao2, regiao3)
  beta_ind <- beta_mat[grupo, ]          # seleciona linha do grupo de cada i
  eta <- alpha_vec[grupo] + rowSums(X * beta_ind)
  exp(eta)
}
  11. Geração das contagens

    Com as médias condicionais calculadas, as contagens são amostradas:

    • Consultas, PS, Exames: rnbinom(n, size=θ, mu=μ)
    • Terapias, Internações: componente de zero estrutural + NB
    rzinb_mu <- function(mu, theta, pzero) {
  zero_flag <- rbinom(length(mu), size=1, prob=pzero)
  y_nb      <- rnbinom(length(mu), size=theta, mu=mu)
  ifelse(zero_flag == 1, 0, y_nb)
}
  12. Variáveis derivadas

    Calculadas a partir das contagens:

    • Razões: razao_exames_consultas = exames / (consultas + 1)
    • Gasto: combinação linear com ruído log-normal
    • Meses com utilização: modelo Beta-Binomial (ver seção 8 da metodologia)
    • Prestadores distintos: Poisson com cap no percentil 99
    • Flags: indicadores binários por percentil 95
  13. Montagem da base final

    Todas as variáveis são reunidas em um data.frame de $5.000 \times 21$ com tipos adequados (factor para categóricas, integer para contagens).

  14. Checagem vs. alvos ANS

    Comparação das médias simuladas com os indicadores oficiais. As diferenças percentuais devem ser menores que 5% para validar a calibração:

    comparacao_ans <- data.frame(
  variavel = names(alvos_ans),
  alvo_ans = as.numeric(alvos_ans),
  media_observada = as.numeric(medias_obs),
  diferenca_percentual = 100 * (medias_obs - alvos_ans) / alvos_ans
)
  15. Validação das médias teóricas $\mu$

    Além de comparar com a ANS, o script também valida se as médias teóricas condicionais (as esperanças $\mu_{igk}$ antes da aleatorização) por grupo fazem sentido clínico:

    valid_mu <- data.frame(
  mu_consultas   = tapply(mu_consultas, dados_sim$grupo_latente, mean),
  ...
)
  16. Gráficos e exportação

    Sete gráficos são gerados com ggplot2:

    1. Histograma do total de eventos
    2. Boxplot total de eventos por grupo
    3. Boxplot razão exames/consultas por grupo
    4. Dispersão gasto × eventos por grupo (escala log)
    5. Histograma de meses com utilização
    6. Prevalência de flags por grupo (gráfico de barras)
    7. Boxplot razão PS/consultas por grupo

    A base é exportada como CSV para o caminho definido em write.csv().
